# P6.3: Selective Media Sharing & Batch Delete

**Description:** Allow photographers to select specific media items to share only selected items, or delete multiple items at once.

**Entry Point:** Dashboard - Media Grid selection mode + Share/Delete buttons

---

## Acceptance Criteria

### Selection Mode
- [x] "Select" button toggles selection mode on/off
- [x] Checkbox appears on each media item when in selection mode
- [x] Clicking checkbox toggles selection for that item
- [x] Clicking media item opens lightbox for preview (even in selection mode)
- [x] "Select All" / "Deselect All" buttons when in selection mode
- [x] Selected count displayed in toolbar
- [x] "Done" button exits selection mode

### Selective Sharing
- [x] "Share Gallery" button shows "Share X Selected" when items are selected
- [x] Share link contains only selected media when items are selected
- [x] Share modal shows QR code and sharing options
- [x] Share Links panel shows item count for each link ("X items" or "All media")
- [x] Selection mode exits after sharing

### Batch Delete
- [x] "Delete" button appears in toolbar when items are selected
- [x] Confirmation modal shows count of items to be deleted
- [x] Batch delete removes all selected items
- [x] Success toast shows count of deleted items
- [x] Selection mode exits after deletion
- [x] Gallery refreshes to show updated media

## UI Design

### Dashboard Toolbar (Selection Mode Off)
```
+----------------------------------------------------------+
| Media (24 items)                              [Select]    |
+----------------------------------------------------------+
```

### Dashboard Toolbar (Selection Mode On - No Selection)
```
+----------------------------------------------------------+
| Media (24 items)                                          |
|         [Select All] [Deselect All]           [Done]      |
+----------------------------------------------------------+
```

### Dashboard Toolbar (Selection Mode On - With Selection)
```
+----------------------------------------------------------+
| Media (24 items)              5 selected                  |
|         [Select All] [Deselect All] [Delete]  [Done]      |
+----------------------------------------------------------+
```

### Header Button States
```
No selection:     [Share Gallery]
With selection:   [Share 5 Selected]
```

### Media Grid (Selection Mode)
```
+--------+  +--------+  +--------+
| [x]    |  | [ ]    |  | [x]    |   <- Checkbox in corner
|  IMG   |  |  IMG   |  |  VID   |   <- Click opens preview
|        |  |        |  |        |
+--------+  +--------+  +--------+
  ring       no ring     ring        <- Visual selection indicator
```

### Delete Confirmation Modal
```
+------------------------------------------+
|  Delete 5 Items                       X  |
+------------------------------------------+
|                                          |
|  Are you sure you want to delete 5       |
|  selected items? This action cannot      |
|  be undone.                              |
|                                          |
|              [Cancel]  [Delete 5 Items]  |
+------------------------------------------+
```

---

## Technical Notes

- Selection state is local only (not persisted)
- When selection mode is on:
  - Clicking media item toggles selection
  - Eye button in top-right opens lightbox for preview
  - Checkbox indicator in top-left shows selection state
- Batch delete uses existing single-item delete endpoint in a loop
- Delete from database first, then from storage
- Confirm modal prevents accidental mass deletion
