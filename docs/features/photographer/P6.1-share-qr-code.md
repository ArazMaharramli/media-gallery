# P6.1: Share QR Code

**Description:** Display a QR code in the share modal for easy mobile sharing.

**Entry Point:** Share Gallery modal

---

## Acceptance Criteria

- [ ] QR code displayed prominently in share modal
- [ ] QR code encodes the full gallery URL
- [ ] QR code is scannable by standard phone cameras
- [ ] QR code updates when a new share link is generated
- [ ] Download QR code as PNG image
- [ ] QR code has sufficient size and contrast for easy scanning

---

## UI Design

```
+------------------------------------------+
|  Share Gallery                        X  |
+------------------------------------------+
|                                          |
|         +------------------+             |
|         |                  |             |
|         |    [QR CODE]     |             |
|         |                  |             |
|         +------------------+             |
|                                          |
|  Share this link with your clients:      |
|  +----------------------------------+    |
|  | https://gallery.ex.../abc123    | [] |
|  +----------------------------------+    |
|                                          |
|  [Download QR]  [WhatsApp] [Email] [SMS] |
|                                          |
+------------------------------------------+
```

---

## Technical Notes

- Use a lightweight QR code library (e.g., `qrcode` npm package)
- Generate QR code client-side for instant display
- QR code size: minimum 200x200px for reliable scanning
- Include quiet zone (white border) around QR code
- Error correction level: M (15% recovery) for balance of size/reliability

---

## Dependencies

```
npm install qrcode
```
