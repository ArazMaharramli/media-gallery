# Features

Feature specifications organized by user persona.

---

## Personas

| Persona | Description |
|---------|-------------|
| [Photographer](./photographer/) | Creates and manages events |
| [Client](./client/) | Views gallery via shared link |
| [Guest](./guest/) | Uploads media via shared link |

---

## Permissions Matrix

| Action | Photographer | Client | Guest |
|--------|:------------:|:------:|:-----:|
| Create event | Yes | - | - |
| View dashboard | Yes | - | - |
| Upload media | Yes | - | Yes |
| View gallery | Yes | Yes | - |
| Download media | Yes | Yes | - |
| Delete media | Yes | - | - |
| Generate links | Yes | - | - |
