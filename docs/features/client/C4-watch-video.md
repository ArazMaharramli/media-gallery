# C4: Watch Video

**Description:** Watch a video in embedded player.

**Entry Point:** Click on video in grid

---

## Acceptance Criteria

- [ ] Click video opens video player overlay
- [ ] Native HTML5 video player
- [ ] Play/pause controls
- [ ] Progress bar / seek
- [ ] Volume control
- [ ] Fullscreen option
- [ ] Download button

---

## UI Elements

- Video overlay container
- HTML5 video element with controls
- Custom download button
- Close button
